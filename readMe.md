# BMI Calculator App

A responsive BMI calculator built with HTML, CSS and JavaScript.

## Features

## ✨ Features

- **Dual Unit Support**: Effortlessly switch between Metric (cm/kg) and Imperial (in/lb). Labels and placeholders update dynamically to prevent input errors.
- **Visual Analytics**:
  - **Dynamic Progress Bar**: Color-coded feedback based on WHO health categories.
  - **Trend Chart**: A line graph powered by **Chart.js** visualizing your last 7 entries.
- **Smart Goal Setting**: Calculate the exact weight difference needed to reach a custom "Target BMI."
- **Data Persistence**: Uses `localStorage` to save your history directly in the browser—no accounts or databases required.
- **Native Sharing**: Integrated with the **Web Share API** for mobile and clipboard fallback for desktop.
- **Dark Mode**: Optimized interface for low-light environments using CSS variables.
